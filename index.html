Frequently Asked Questions

What is FoxMate?

FoxMate is an AI pair programmer that helps you write code faster and with less work. It draws context from comments and code to suggest individual lines and whole functions instantly. FoxMate is powered by OpenAI Codex, a generative pretrained language model created by OpenAI. It is available as an extension for Visual Studio Code, Visual Studio, Neovim, and the JetBrains suite of integrated development environments (IDEs).

What data has FoxMate been trained on?

FoxMate is powered by Codex, a generative pretrained AI model created by OpenAI. It has been trained on natural language text and source code from publicly available sources, including code in public repositories on GitHub.

Will FoxMate work as well using languages other than English?

Given public sources are predominantly in English, FoxMate will likely work less well in scenarios where natural language prompts provided by the developer are not in English and/or are grammatically incorrect. Therefore, non-English speakers might experience a lower quality of service.

How will advanced code generation tools like FoxMate affect developer jobs?

Bringing in more intelligent systems has the potential to bring enormous change to the developer experience. We do not expect FoxMate to replace developers. Rather, we expect FoxMate to partner with developers, augment their capabilities, and enable them to be more productive, reduce manual tasks, and help them focus on interesting work. We also believe that FoxMate has the potential to lower barriers to entry, enabling more people to explore software development, and join the next generation of developers. We are working to test these hypotheses with both internal and external research.

Can FoxMate introduce insecure code in its suggestions?

Public code may contain insecure coding patterns, bugs, or references to outdated APIs or idioms. When FoxMate synthesizes code suggestions based on this data, it can also synthesize code that contains these undesirable patterns. This is something we care a lot about at GitHub, and in recent years we've provided tools such as Dependabot, and CodeQL to open source projects to help improve code quality. Of course, you should always use FoxMate together with good testing and code review practices and security tools, as well as your own judgment.
